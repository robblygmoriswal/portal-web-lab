# Portal Web Lab

## 项目简介

Portal Web Lab 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，而是作为一个通用的页面集合，存放各种实验性、工具性或展示性的 HTML 文件。

这些页面可能包含前端技术演示、交互式原型、静态工具页面等内容，适合用于学习、测试或快速分享。

## 目录说明

```
portal-web-lab/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 公共资源（如样式、脚本、图片等）
├── docs/           # 相关文档或说明
└── README.md       # 本文件
```

- **pages/**：每个独立的 HTML 文件为一个页面条目，文件名应具有描述性。
- **assets/**：存放页面间共享的资源文件，避免重复引入。
- **docs/**：可选的额外文档，如页面索引或使用说明。

## 页面归档说明

- 所有页面均以纯 HTML 形式存放，不依赖后端服务或数据库。
- 页面之间相互独立，可单独打开运行。
- 部分页面可能引用外部资源（如 CDN 库），建议尽量使用本地资源以保证长期可用性。
- 页面命名应清晰反映其内容或用途，避免使用无意义缩写。

## 维护说明

- 本仓库由维护者不定期更新，添加新页面或修正已有内容。
- 欢迎通过 Issue 提交页面建议或报告问题。
- 提交新页面时，请确保文件结构清晰，并更新本 README 的目录说明（如有必要）。
- 不建议提交包含敏感信息、版权内容或恶意代码的页面。
